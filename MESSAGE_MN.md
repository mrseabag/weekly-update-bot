# Долоо хоногийн тайлангийн бот — тайлбар (Монгол)

## Энэ юу вэ, хэрхэн ажилладаг вэ?

Энэ бол Telegram дээр ажилладаг автомат бот бөгөөд багийн гишүүдээс долоо хоног бүр
ажлын тайлан авч, нэгтгэн менежерт илгээдэг.

- **Даваа гараг бүр өглөө 9 цагт (UTC)** бот бүртгэлтэй бүх гишүүнд мессеж илгээж,
  тухайн долоо хоногийн тайлангаа бөглөхийг хүснэ.
- Гишүүн бүр 3 зүйлд хариулна:
  1. Энэ долоо хоногт юу хийж дуусгасан бэ?
  2. Одоо юу дээр ажиллаж байна вэ?
  3. Холбогдох файл (баримт бичиг, зураг, видео г.м.) хавсаргах — заавал биш
- Хариултууд Excel хүснэгтэд автоматаар, цэгцтэй хадгалагдана.
- **Баасан гараг бүр оройн 5 цагт (UTC)** энэ хүснэгтийг менежерийн имэйл рүү
  автоматаар илгээнэ.

Хэрэглэгч хүссэн үедээ /update командаар давхар тайлан илгээж болно.

## Ямар мэдээлэл дамждаг вэ?

- Хэрэглэгчийн нэр, Telegram-ий хэрэглэгчийн нэр (username) болон Telegram-ийн дотоод ID
- Бичсэн 2 хариулт (юу дуусгасан, юу дээр ажиллаж байгаа)
- Хавсаргасан файл (байвал)

Энэ мэдээлэл зөвхөн (1) ботыг ажиллуулж буй серверийн Excel файлд, мөн (2) менежерийн
имэйл хайрцагт хадгалагдана. Өөр ямар нэгэн гуравдагч этгээд, компанид дамждаггүй.

## Хэрхэн аюулгүй ажилладаг вэ?

- Ботын нэвтрэх түлхүүр (bot token), имэйлийн нэвтрэх нууц үг зэрэг нууц мэдээллийг
  кодын файл дотор бичдэггүй — тусад нь, орчны хувьсагч (environment variable)
  байдлаар хадгалдаг тул кодыг хэн нэгэнтэй хуваалцсан ч нууц мэдээлэл алдагдахгүй.
- Имэйл илгээхэд Gmail-ийн тусгай "App Password" ашигладаг тул жинхэнэ Gmail
  нууц үг хэзээ ч хадгалагддаггүй, харагддаггүй.
- Telegram-тай холбогдох бүх харилцаа Telegram-ийн албан ёсны, шифрлэгдсэн
  (encrypted) Bot API-гаар дамждаг.
- Менежерт очих имэйл SSL шифрлэлттэй холболтоор (SMTP over SSL) явдаг.
- Бот cloud hosting үйлчилгээ дээр 24/7 ажиллах бөгөөд зөвхөн тохиргоо хийсэн
  админ хүн серверийн өгөгдөлд хандах эрхтэй.
